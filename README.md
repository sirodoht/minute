# minute

Self-hosted uptime monitor.

## Usage

```sh
$ minute sites.txt
```

## Configuration

Configuration is done through the `sites.txt` file.

```
<SMTP server URL>
<SMTP username>
<SMTP password>
<From email>
<To email>
<Website 1>
<Website 2>
...
<Website n>
```

Check [`sites-example.txt`](sites-example.txt) for an example.
